# VsCode + 단축키
* 주석 : command(⌘) + /
* 여러줄 주석 : shift(⇧) + option(⌥) + a
* 파라미터 보기 : command(⌘) + shift(⇧) + 스페이스바
* 블록 단위 접기, 펼치기 : option(⌥) + command(⌘) + [ or option(⌥) + command(⌘) + ]
* 에러 해결 추천 받기 : command(⌘) + .
* 한 문장 위아래 이동 : option(⌥) + ↑ or option(⌥) + ↓
* 한 라인 복사 : option(⌥) + shift(⇧) + ↑ or option(⌥) + shift(⇧) + ↓
* 동일한 모든 단어 선택 : command(⌘) + shift(⇧) + l
* 이름 바꾸기 : fn + F2
